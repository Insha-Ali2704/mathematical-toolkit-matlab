# mathematical-toolkit-matlab
Mathematical Toolkit in MATLAB

This project is a menu-driven MATLAB application that performs:

• Differential Equation solving  
• Function plotting  
• Mathematical models (growth, decay, logistic, etc.)  
• Curve fitting (linear, polynomial, exponential)  
• Matrix operations (inverse, determinant, eigenvalues, etc.)

The toolkit allows users to input values and visualize results interactively.

Skills used:
MATLAB, Mathematical Modelling, Numerical Methods
